It shows the number of Twitter followers, the number of friends and the number of updates for a Twitter user. <img style="border-right-width: 0px; display: block; float: none; border-top-width: 0px; border-bottom-width: 0px; margin-left: auto; border-left-width: 0px; margin-right: auto" title="image" border="0" alt="image" src="/wp-content/uploads/files/media/image/WindowsLiveWriter/StickyPluginforSeesmicDesktop2_14FAD/image_8.png" width="281" height="273" /></p>  <p>&#160;</p>  <p>Here is a zoomed in close up of this information and the icon. When creating an icon you can customize the collapsed text as makes sense for your plugin.&#160; </p>  <p><img style="border-right-width: 0px; display: block; float: none; border-top-width: 0px; border-bottom-width: 0px; margin-left: auto; border-left-width: 0px; margin-right: auto" title="image" border="0" alt="image" src="/wp-content/uploads/files/media/image/WindowsLiveWriter/StickyPluginforSeesmicDesktop2_14FAD/image_7.png" width="247" height="71" /></p>  <h3>Expanded State</h3>  <p>In its expanded state, the Sticky plugin hides the collapsed view and transitions to a larger view. The expanded state shows the user’s friendly name, a link to their web page, a link to the TwitterCounter for the user, a link to their Twitter page, and the user’s Twitter profile bio (see below).</p>  <p><img style="border-right-width: 0px; display: block; float: none; border-top-width: 0px; border-bottom-width: 0px; margin-left: auto; border-left-width: 0px; margin-right: auto" title="image_16" border="0" alt="image_16" src="/wp-content/uploads/files/media/image/WindowsLiveWriter/StickyPluginforSeesmicDesktop2_14FAD/image_16_3.png" width="294" height="348" /></p>  <p>&#160;</p>  <p>&#160;</p>  <p>&#160;</p>  <p>&#160;</p>  <p>I used some visual states to subtly animate the sticky note when you move your mouse over it. This is a pretty simple use of visual states, easing, and effects which I’ll explore in an upcoming post when I dive into the details of how the plug in is created.</p>  <h3>Choose Your Stick Color</h3>  <p>The Sticky plugin now allows you to choose a custom color for the Sticky note. First, go to the Settings for the Sticky plugin. Find the Sticky plugin in the list and click the Settings link. <img style="border-right-width: 0px; display: inline; border-top-width: 0px; border-bottom-width: 0px; border-left-width: 0px" title="image" border="0" alt="image" src="/wp-content/uploads/files/media/image/WindowsLiveWriter/StickyPluginforSeesmicDesktop2_14FAD/image_13.png" width="594" height="457" /> </p>  <p>Then you can set the color or revert to the default color. <img style="border-right-width: 0px; display: inline; border-top-width: 0px; border-bottom-width: 0px; border-left-width: 0px" title="image" border="0" alt="image" src="/wp-content/uploads/files/media/image/WindowsLiveWriter/StickyPluginforSeesmicDesktop2_14FAD/image_19.png" width="590" height="454" /> </p> <a name="dload"></a>  <h3>Set the Follower Count Threshold<font color="#ff0000"><strong> - NEW</strong></font></h3>  <p>You can set the follower count threshold in the settings, too. It defaults to a value of 100 and be be adjusted from 0 to 10,000. This setting turns Sticky off for any timeline items where the follower count is less than the threshold.</p>  <h3>Download It</h3>  <p>You can download this plugin from the link below.
