<p></p>  <p>I created a simple user control using the Popup control as a starting point. The end result is a nice little user control that can be used as a popup control to display a message. It has only 1 button on it (an OK button that closes the popup control). I’ll outline the basic steps of creating the User Control and how the popup control works with it. Of course, you can create a popup control without the user control, but its a much cleaner practice to create user controls. </p>  <p><strong>Step 1)&#160; </strong>Create a tester control. I called mine PopupTester.xaml. It can contain just about anything you want, in my case I put a funky looking green filled border on the control and a button, that when clicked will call the user control that we’re about to create. You can skip the code for the event handler for the button for now. </p>  <p>The PopupTester control is the default control that will be loaded when the Silverlight application is started. (Set this in App.xaml.cs)</p>  <p>Here is how the PopupTester control looks in Expression Blend. (It has no background color, which is why it appears gray.)</p>  <p><a href="/wp-content/uploads/files/media/image/WindowsLiveWriter/CreateaSilverlight2UserControlfromaPopup_BE63/image_2.png"><img title="image" style="border-right: 0px; border-top: 0px; border-left: 0px; border-bottom: 0px" height="137" alt="image" src="/wp-content/uploads/files/media/image/WindowsLiveWriter/CreateaSilverlight2UserControlfromaPopup_BE63/image_thumb.png" width="244" border="0" /></a> </p>  <p><strong>Step 2)&#160; </strong>Create a new User Control and name it MyPopupControl.&#160; The complete XAML for the control is shown a little further down. Remove the background, width, and height from the Layout Grid control so it only has a name property of LayoutRoot..</p>  <p><strong>Step 3)&#160; </strong>Slap a Popup control in the main layout Grid. Name the popup control popMessage. </p>  <p><strong>Step 4)&#160; </strong>Create a Grid inside of the Popup control. Create 3 rows and 3 columns in the Grid. Make the 2nd row and the 2nd column (the middle ones) have a Width = Auto. This creates a nice little grid that we can use to center the popup control contents. There are other ways to do this, but let’s swing with it.</p>  <p><strong>Step 5) </strong>Add a Border to the Grid inside of the Popup control. Style the border however you wish, but make sure you give it a background color of some sort so it gets filled in. I gave mine a gradient background color from top to bottom and set the opacity to make it slightly transparent.</p>  <p><strong>Step 6) </strong>Next,&#160; add a StackPanel inside of the Border, and stick a TextBlock and a Button inside of the Stackpanel. The TextBlock contain the message and the Button will be used to close the popup control. Name the TextBlock tbPopupMessage and the Button btnClose.</p>  <p>The XAML should look something like this:</p>  <div>   <div style="padding-right: 0px; padding-left: 0px; font-size: 8pt; padding-bottom: 0px; overflow: visible; width: 100%; color: black; border-top-style: none; line-height: 12pt; padding-top: 0px; font-family: consolas, &#39;Courier New&#39;, courier, monospace; border-right-style: none; border-left-style: none; background-color: #f4f4f4; border-bottom-style: none">     <pre style="padding-right: 0px; padding-left: 0px; font-size: 8pt; padding-bottom: 0px; margin: 0em; overflow: visible; width: 100%; color: black; border-top-style: none; line-height: 12pt; padding-top: 0px; font-family: consolas, &#39;Courier New&#39;, courier, monospace; border-right-style: none; border-left-style: none; background-color: white; border-bottom-style: none"><span style="color: #0000ff">&lt;</span><span style="color: #800000">UserControl</span> <span style="color: #ff0000">x:Class</span><span style="color: #0000ff">=&quot;SLPopupDemo.MyPopupControl&quot;</span></pre>
